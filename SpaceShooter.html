Galaxy Shooter

This was the first non-terminal game I made. All of my game before this were made in Python and they were played only with the terminal. I made Galaxy Shooter using the game engine Unity and this gave me my first exposure to C#. This was a fun project to work on and helped me get exposure to the fundamentals of using a game engine.

Features include: music, SFXs, a high score system, and visual damage.
